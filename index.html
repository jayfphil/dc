Get started on Testing Wakanda API A.K.A ICCBS

Step 1

Go to the Globe WFM dev environment and log as any user.

https://globetelecominc-dev.outsystemsenterprise.com/WFM_BackOffice/DashboardEntry.aspx

Step 2

Copy the link below and replace it to your current browser (Actually I didn't add any button or link from the menu screen to redirect you to the API testing screen, The QAs might surprise there is misplace button).

https://globetelecominc-dev.outsystemsenterprise.com/WFM_BackOffice/SaveAPItoEntities.aspx

Step 3

Now there are three input fields and three buttons on your current screen. Just fill up the Work Order field and click the Get Details to trigger the API call from ICCBS and redirect you to the results page.

For Check All Details button : you will be redirect to the results page with all the cache Work Order data.

For Delete button : you need to fill up the Work Order field to delete its Work Order detail cache.

Check the image below if you are currently in the same page.

Step 4

All done! you will be redirected to another page with the Work Order details and information just like the image below. Please check and compare this data from Wakanda/ICCBS entities. Happy Testing :)
